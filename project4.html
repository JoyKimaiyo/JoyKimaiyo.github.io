SQL Portfolio Project: Analyzing Digital Marketing Metrics

By Joyce Kimaiyo | 6 min read

Business Question

This machine learning regression analysis project utilizes the Toyota Corolla CSV dataset to predict the price of a Toyota Corolla based on various features, including Age, KM, FuelType, HP, MetColor, Automatic, CC, Doors, and Weight.

Regression Analysis Overview

Regression analysis is a predictive modeling technique that examines the relationship between a dependent (target) variable and independent (predictor) variables. It is primarily used for predicting continuous variables.

Libraries Used

Pandas: Data analysis, cleaning, exploration, and manipulation. NumPy: Handling arrays and numerical computations. Matplotlib & Seaborn: Data visualization.

Exploratory Data Analysis (EDA)

EDA is essential for understanding the dataset, identifying patterns, detecting anomalies, and validating assumptions. The dataset is already clean, with no null values or unnecessary columns. Distribution analysis is performed using visualizations.

Encoding Categorical Data

Since machine learning models require numerical inputs, categorical variables need to be encoded. One-hot encoding is applied to the FuelType column, replacing it with three new columns. One dummy column is dropped to prevent the dummy variable trap, ensuring that only n-1 categorical variables are used.

Feature Selection and Data Splitting

The independent variables (X) influence the dependent variable (Y), which is the car's price. The dataset is split into training and testing sets for model evaluation.

Linearity and Multicollinearity Check

A good predictive relationship between the independent and dependent variables is necessary. Multicollinearity is checked using a correlation matrix, ensuring that correlation coefficients remain below 0.80, as recommended by NCBI.

Modeling Approaches

The following machine learning models are implemented: Linear Regression Decision Tree Regressor Support Vector Machine (SVM) Regressor Random Forest Regressor

Model Evaluation

The performance of each model is assessed using the Root Mean Squared Error (RMSE), which measures the average difference between predicted and actual values. A lower RMSE indicates better model performance. RMSE Results
SVM: 3753.82 (Highest, least accurate)
Random Forest: 1266.99 (Lowest, most accurate)
Decision Tree: 1610.01
Linear Regression: 1422.64

The Random Forest model achieves the lowest RMSE, indicating it provides the most accurate predictions. Its ensemble approach, which combines multiple decision trees, reduces overfitting and enhances predictive accuracy. The SVM model has the highest RMSE, making it the least effective predictor in this scenario.
